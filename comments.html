Technical Aspects

I have used Flexbox in order to make my website responsive. There are currently three boxes inside the flexbox container two which are invisible and the main box which has all the content.
The two invisible boxes are positionned at the top and at the bottom of the main box, they are used for keeping the box shadow of the main box from going through the top of the page so their are only here for aesthetic purposes.
As for the website's structure it is a basic structure where there is a navigation bar to go from page to page.
For the CSS selectors and Ids, I chose most of them for aesthetic purposes, to change font colour, to change size, to align content. An exemple of Id that I used would be the "current" Id which shows the page you are on currently by indicating it with a blueviolet colour in the navigation bar. I chose to use this as an Id because it is used only once in each pages.

Aesthetics

My website is meant to look like a window from a NES game so I used a Pixelated font and only lines to separate the navigation bar from the rest of the page.
I got my inspiration from two specific games, Mother 2 also known as Earthbound and TokiMeki Memorial. I based the background on Tokimeki Memorial's background and the window aesthetics on Earthbound's window.
The navigation bar is also interactive, as when the cursor hovers on one of the links, the background lights up and the text is underlined. The current page is also indicated by the link with a blueviolet colour.

Accessibility Aspects

Here are some features of my website that are designed for accessibility :

  • Responsivity : My website is responsive so it can be seen by both normal sized screen and small sized screen.
  • Vision impaired friendly : My website has alt text associated with the pictures for those who have to use screen readers.
